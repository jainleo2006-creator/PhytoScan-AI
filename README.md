# 🌿 PhytoScan AI

AI-powered Plant Disease Detection System using TensorFlow and Streamlit.

## Features

- Detects plant diseases
- Supports:
  - Tomato
  - Potato
  - Bell Pepper
- Neural network disease prediction
- Interactive futuristic UI
- Leaf validation system

## Technologies Used

- Python
- TensorFlow
- Streamlit
- OpenCV
- NumPy
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Model

Custom trained MobileNetV2 model for plant disease classification.
